# 双人小餐桌

这是一个手机优先的 PWA 原型。目前版本先实现可用的本地模拟闭环：冰箱、今日餐桌、菜谱、购物清单、双人贴纸头像，以及“番茄炒蛋”从菜谱到餐桌再回到冰箱的流程。

## 本地预览

最简单可以直接打开 `index.html`。如果要测试 PWA 外壳，建议在项目目录启动一个静态服务器：

```bash
python3 -m http.server 5174
```

然后打开：

```text
http://localhost:5174
```

如果端口被占用，换成任意空闲端口即可。

## 当前范围

- 三个底部导航页面：我的冰箱、今日餐桌、我的菜谱
- 冰箱结构：冷冻上下层、保鲜三层、保鲜抽屉、下层门架三层
- 添加和编辑食材，调整数量，移动存放区域，购物清单买到后入库
- 今日餐桌支持本周七天选择，可查看当天或本周概览
- 每一餐都可以添加菜：从菜谱选择，或写临时菜名
- 菜谱可以安排到任意一天的早餐、午餐、晚餐或加餐
- 新建菜谱支持录入菜品照片、标签、所需食材和步骤
- 菜谱详情可给每道菜添加或更换做饭照片
- 番茄炒蛋闭环：安排到餐桌、食材检查、做菜模式、吃完确认、扣减库存、剩菜回冰箱、菜谱历史
- 双人贴纸头像可换颜色、形状、发型和表情
- 冰箱门便签和贴纸可编辑
- 添加食材的类别选择使用横向滑动贴纸，不再显示独立单位输入框
- 冰箱插画中每个区域最多直接显示 3 个食材贴纸，超过后用 `+N` 气泡收起
- 内置轻柔合成音效默认开启，可手动关闭
- 小唱片机入口默认开启 soft jazz 氛围音乐，也可以拖入本地音频临时播放
- 页面、弹窗、冰箱门和卡片点击有轻量过渡动画
- 清爽留白的手绘生活绘本方向
- iPhone Pro / Pro Max 优先的移动端布局
- PWA manifest 和 service worker 外壳

## 插画和音频素材

冰箱主图已经接入开门和关门两张状态图，食材品类贴纸放在 `assets/food/`。后续如果生成了更合适的手绘素材，可以在 `app.js` 顶部的 `illustrationAssets` 和 `categoryAssets` 中替换路径：

```js
const illustrationAssets = {
  fridgeOpen: "./assets/illustrations/fridge-open.png",
  fridgeClosed: "./assets/illustrations/fridge-closed.png",
  table: "./assets/table.png",
  recipe: "./assets/recipe-book.png"
};
```

当前短音效和 soft jazz 背景音乐使用 Web Audio 在浏览器内生成，不依赖外部音频文件，也没有版权素材风险。后续如果确定使用 Kenney、Mixkit 或其他授权明确的音频文件，可以替换为真实音频资源。
