# 双人小餐桌视觉语言草案

## 核心气质

整体方向是温暖但清爽的手绘生活绘本，不做标准白底卡片 App。界面应像一本被两个人持续使用的家庭饮食手账：纸张、便签、木牌、菜谱卡、手绘冰箱、餐桌、锅和盘子都来自同一个绘画世界。画面要留白，不要塞满，也不要偏黄油腻。

## 颜色

- 背景：奶白、浅纸浆色、淡灰蓝或淡灰绿，不使用冰冷纯白，也不大面积使用浓黄。
- 主色：墨绿色、灰蓝色、木色作为稳定生活感。
- 点缀：砖红色和少量芥末黄，只用于番茄、便签、提醒和互动状态。
- 禁忌：高饱和糖果色、光滑渐变、玻璃透明高光、科技蓝紫霓虹。

## 线条与形状

- 轮廓使用深色铅笔线，不追求几何完美。
- 圆角和边缘允许轻微不规则，像手剪纸片或手画卡片。
- 图标采用水粉/彩铅贴纸感：低饱和填色、圆头描边、轻微变形。
- 点击区域仍需足够大，不能因为手绘感牺牲可用性。
- 两个共同管理人的默认头像都保持中性/男生贴纸方向，不默认画成一男一女。

## 组件语言

- 底部导航：三张手绘纸标签或小木牌。
- 弹窗：便签纸、食谱卡片、打开的抽屉。
- 冰箱：一件手绘家具，门和内部隔层都要有物件感。
- 插画资产：代码中保留 `illustrationAssets.fridge/table/recipe` 三个替换点，后续生成图只替换资产路径，不改变页面结构。
- 今日餐桌：绘本里的餐桌、盘子、锅、餐垫，不使用表格日历。
- 我的菜谱：索引卡、手账本、食谱纸，而不是资料库列表。

## 动效

- 动画像绘本物件轻轻移动：冰箱门打开、便签弹出、食材落到盘子里。
- 避免科技感的滑入、玻璃模糊、发光、强弹跳。
- 所有动画都应短、轻、可忽略，不影响快速操作。
